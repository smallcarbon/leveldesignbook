关键路径

the minimum / main player path to complete a single player level

这 关键路径 （或 黄金之路）是玩家完成关卡和进度的主要路径/程序。

  • usually for single player levels with scripted progressions, not multiplayer

  • 突出显示关卡中最重要（“关键”）的部分。

  • 代表一个 理想的玩家流程 忽略可选的侧面区域

最后一点很重要。关键路径代表您的 理想的 design goals and requirements, but it is not the reality.大多数玩家会徘徊、探索、徘徊，甚至只是感到困惑。

如何规划关键路径

In a layout drawing, the critical path markup helps other designers and collaborators understand the level's 踱步。发生了什么以及在哪里发生？

  1. 画一个首字母 布局.

  2. Mark and label where the player starts and exits.

  3. 对任何专业进行标记、标签和编号 节拍.

  4. 如果看起来不会太混乱，那就用箭头画出玩家的路线。

有关绘制平面图的更多信息，请参见 布局.

作为范围界定工具的关键路径

在工程中，项目经理使用 关键路径法（CPM） 找出哪些任务必须首先发生以及为什么。绘制关键路径有助于他们理解项目的逻辑和流程。

We can use critical paths for level design in a similar way, to help manage project 范围.

  • 该关卡的每个部分有多重要？

  • If something isn't on the critical path, then only some players will see it.值得制作吗？

关键路径帮助我们想象“最小可行”布局是什么样的。

For example, in the layout drawing above, the yellow hatched area is not on the critical path.那么它还值得制作吗？如果我们削减这个领域，那么我们要做的工作就会减少。

Maybe the level is still too big and too much work.也许我们需要删除更多。

如果我们删除所有侧面区域会怎么样？ Does it still fulfill our experience design goals? Maybe we need to redesign the critical path entirely...

反对关键路径？

Designing around a critical path is potentially a reductive way for thinking about a space.

关键路径通过资源、大门和目标来工具化世界，这是关卡设计师强加给玩家的活动清单。游戏和关卡能否不仅仅是供用户消费的“内容”？如果您的项目需要高度的真实性、合理性或地方感怎么办？围绕关键路径构建通常会产生“电子游戏”的感觉空间。

Instead, some designers prefer to assemble a critical path 之后 一个已经完成的 布局 和 白盒，围绕现有的关卡几何体流动。 Nintendo 64 的 Goldeneye 007 (1997) 制作人 Martin Hollis 解释了他们的关卡设计过程：

“关卡创建者或建筑师在没有太多关卡设计的情况下进行工作，我的意思是，他们通常没有考虑到玩家的起点或出口。当然，他们没有考虑敌人的位置或物体的位置。他们的工作只是创造一个有趣的空间。关卡制作完成后，戴夫或有时邓肯将面临着在其中填充目标、敌人和东西。这种草率的计划外方法的好处是，游戏中的许多关卡都具有现实和非线性的感觉。有些房间坦率地说，关卡中有多种路线，这种设计方法效率很低，因为大部分关卡对于游戏玩法来说是不必要的，但它有助于增强游戏的自由感，而这种自由感和真实感反过来又对游戏的成功做出了巨大贡献。” ——Martin Hollis，《黄金眼 007》（N64）的制片人，引述于 “《黄金眼 007》中的反设计/向后游戏设计”作者：Chris DeLeon

关键路径示例：死亡循环 (2021)

在第一人称角色扮演游戏死亡循环 (2021) 的“复杂”关卡下图中，关卡设计师 Sylvain Menguy 强调了非线性单人游戏地图的关键路径。他们记录了下午的配置和晚上的配置，以及玩家到达区域老板的可能路径。

游戏有4张主地图，可以在一天中的4个时间段（早上、中午、下午和晚上）进行游戏，玩家可以在其中自由探索关卡、进行战斗、发现谜题，甚至解决构成游戏主要冒险的“谋杀谜题”……

这个近似的布局图省略了不必要的细节，有利于大局观，这对于沟通很有用 踱步。请注意，下面显示的白色箭头与地图地形并不完全匹配，但仍然传达了有关预期路线的大量信息。尽管玩家一开始有 3 条车道，但这些车道始终会汇聚到地图另一侧 Boss 竞技场的两个入口。

该地图从一开始就是为了允许多人游戏而制作的，另一个玩家会在下午或晚上入侵。因此，这两个时期对布局的解读有所不同，与前来追赶柯尔特并让他过得更艰难的对手朱莉安娜的遭遇战也有所不同。

这种时间段之间的共享布局总是给出了多种导航选择，禁止出现死胡同和长廊，让你的体验更加流畅，让你在区域之间快速轮换……”

- 西尔万·孟盖

“旋转”是指玩家必须将焦点（“旋转”）转移到另一个地图区域，通常是在《反恐精英：全球攻势》等基于团队的多人射击游戏中。有趣的是，Menguy 在单人关卡中采用了多人关卡设计思维。

有关多人游戏设计中旋转的更多信息，请参阅 地图平衡.

要审查...

这 关键路径 是通过单人关卡的理想玩家路线。

它主要是一个规划和范围界定工具。它有助于沟通 踱步 在一个 布局 绘图。如果您需要缩小项目范围，它可以帮助您决定要削减关卡的哪些部分。

然而，优先考虑关键路径可能会导致更多人为的感觉空间。有些关卡设计师先设计一个空间，然后再找出关键路径。
