☀️灯光

如何使用功能性照明策略照亮游戏关卡

灯光设计 是在游戏世界中放置灯光来创造视觉深度、唤起情绪并提供信息来帮助玩家玩游戏。光线不足的空间常常让人感觉陈旧、单调、未完工或混乱。

由于照明对于可读性非常重要，因此我们建议在建立核心游戏玩法期间或之后不久进行照明通行证 白盒 或 脚本编写 阶段。

业界通常将照明归类为 环境美术，但在这里我们将较少关注图形，而更多关注 设计 和 功能 照明。 灯说什么、做什么？

在哈雷·威尔逊的上述研究中， 灯光告诉我们很多关于游戏世界的信息并帮助我们玩游戏：

  • 世界构建。沿着墙壁的嵌入式筒灯让人想起一个别致的现代画廊。

  • 体量。较冷的灰色照亮了画廊的墙壁，较温暖的灯光照亮了柜台，昏暗的筒灯标志着阴暗的角落。灯光将这个区域分为三个子区域。

  • 循环动线。远处的墙很黑，也许通向后门或壁橱。它可能不是主要出口，它的照明会更明显。

什么是视频游戏照明？

在现实生活中， 光 是与表面相互作用的可见能量。这是一个优雅的系统；利用能量和光子的核心物理原理，您可以导出所有其他复杂的光效。

相比之下， 视频游戏灯光并不优雅。 这是一个假得一塌糊涂的烂摊子，由游戏引擎中的许多不同子系统秘密组成：

  • 光源： 直接照明的基础层，基于光角度和位置

  • 阴影： 对象遮挡（阻挡）光线并将阴影贴图投射到其他对象上

  • 材料： 3D 表面的颜色、纹理、不透明度和反射率

  • 后处理： 屏幕空间效果，例如色彩校正、光晕和 HDR 眼睛适应

  • 反思：真正的反射率需要重新渲染场景；大多数游戏都使用近似值

  • 烘烤： 开发人员预渲染光照贴图和反射数据

想象一下用棱镜制作彩虹。在现实生活中，您通过棱镜照射一束光线，彩虹“自动”出现，因为这就是光子和物理的工作原理。

但在视频游戏中，棱镜效果将是一团乱麻：

  • 折射玻璃材质+屏幕空间反射棱镜形状

  • Ray FX、白色光束和彩虹在 Photoshop 中分别绘制

  • 光束与棱镜相交处的发光精灵

  • 附近表面的焦散投影/半透明 AO

这不是一个光子与表面相互作用的优雅宇宙。在上面的列表中，请注意我们甚至没有使用任何光源！都是“假”的！

点亮视频游戏就是只挑选你需要的东西。 在大多数游戏引擎中，您可以设置一盏灯来关闭阴影，或设置另一盏灯来禁用反射率。这对于 优化.

这足以让现实生活中的物理学家呕吐。 “但是没有阴影或反射率的光没有任何意义，”物理学家喊道。哈哈，愚蠢的物理学家！这是电子游戏之地！

图形程序员和物理学家一样暗自烦恼。光线追踪等技术旨在让游戏照明更加优雅。也许10年后就会这样。

光的简史

太阳和月亮（反射太阳光）是最常见的 自然光源.

还有 人造光源 例如受控火、煤气灯、白炽灯泡……而在 21 世纪，出现了节能荧光灯和 LED 照明。

然而，这不仅仅是一个技术和进步的故事。灯泡并没有让太阳变得过时，LED 也没有让火变得过时。我们仍然喜欢阳光明媚的日子和浪漫的烛光晚餐。

火并没有从世界上消失。相反，火的含义和用途发生了变化。照明设计就是要了解光如何传达这些想法、情绪和情感。

什么是灯光设计？

现实生活中的灯光设计 是一门根据环境和功能放置光源的艺术/科学，无论是办公室还是喜怒无常的餐厅。

现实生活中的专业照明设计师评估建筑规划并与建筑师协调。他们购买灯具制造商发布的目录，其中包含详细的技术规格和经过实验室测试的光衰减图表/标准化 IES 配置文件，他们可以在 3D 模拟中进行测试。他们还必须平衡有关最低光照水平、预算、维护计划、能源使用和可持续性的当地法律和建筑规范。

由于眼睛会适应附近的光线水平，因此很难猜测房间的实际亮度，因此照明设计师通常会使用 管腔/带状空腔法。为了更准确地现场读数，他们还使用手持式 测光表 测量科学单位，例如 勒克斯 / 流明 / 坎德拉.

一些游戏引擎越来越多地采用这种技术/科学照明方法。然而，我们必须保持警惕： 现实生活中的照明设计与视频游戏照明设计有着根本不同的目标。

光源

由于游戏光照非常复杂，我们将主要关注关卡设计师的光照设计基础知识： 放置光源。

  • 核心光源类型

  • 静态光源与动态光源

  • 灯具与光源

核心光源类型

几乎每个游戏引擎都使用这四种核心光源类型：

  • 环境光 是世界上默认的最小光量。不太现实。

  • 定向光 从一个方向投射持续的光，就像从天空向下投射的阳光一样。

  • 点光源 （全向/“全向”光）就像灯泡一样，向各个方向投射光。

  • 聚光灯 从某个位置向某个方向投射一束光锥。

聚光灯

其他光形状只是这些核心类型的变体。

  • 区域灯为宽扁矩形聚光灯

  • 管灯是长点灯

  • 发射材料/“纹理照明”使用像点光源一样的自发光像素

静态与动态

一个 静态光 根本没有改变，而 动态光 可以改变颜色、强度、方向或位置。

静态光照对于帧率来说几乎总是更好，因为引擎可以“烘焙”光照贴图和反射等光照数据，但重新烘焙不断变得烦人，而且光照数据会消耗大量内存。

相比之下，动态光意味着关卡可以改变并做出反应，从而实现可切换的移动灯和昼夜循环——但如果光源太多，则可能会使玩家的机器过载，帧率也会受到影响。

灯具与光源

一个 灯具 是一种可见的、合理的光源，如灯泡或壁炉。但有阳光的窗户也像一个固定装置。在游戏关卡中，这通常类似于建筑特征或装饰灯道具。

这与发动机内光源不同，后者实际上是一种没有明显原因的不可见光源。它通常看起来有点奇怪，并且在现实生活中根本不会发生。

一个 动机之光 是一个具有合理固定装置的光源。

请注意，一个 单个灯具实际上可以激发多个不可见的发动机内光源。 在哈雷·威尔逊拍摄的下图中，只有两个可见的灯具：橙色的壁炉和蓝灰色的窗户。 但看似两盏灯，实际上是11+个光源！

照明理论

以下是实现照明设计的两种方法：

  • 三点照明： 电影和摄影等 2D 媒体的通用理论

  • D6照明： 对于建筑师和室内设计师来说更抽象但更 3D

三点布光

三点布光 是一种具有三种类型灯光的照明理论：

  • 主灯： 主光源

  • 补光灯： 更柔和的辅助光源来提亮阴影

  • 边缘光： 强调突出物体和边缘

虽然这些核心概念很有用，但它对于关卡设计有局限性。这项技术来自摄影和电影，因此它假设线性固定 2D 屏幕构图，而不是观看者自由导航的交互式 3D 空间。

有关 2D 照明理论的更多信息，请参阅 三点布光.

D6 灯光法

D6 灯光法 是一种专注于覆盖范围和空间的照明理论，受到六面模具的启发。每面代表一种照明策略，总共有六种策略：

  1. 焦点⚀

  2. 焦框⚁

  3. 路径 ⚂

  4. 面积 ⚃

  5. 有焦点的区域 ⚄

  6. 有路径的区域 ⚅

该理论在建筑和关卡设计方面有更多应用，但它比三点理论晦涩难懂且鲜为人知。

有关 3D 照明理论的更多信息，请参阅 D6 灯光法.

如何点亮关卡

照明可能非常耗时，因此不要尝试一次完成最终照明。相反，应在工作流程中建立灵活性并迭代工作。

我们建议分四遍进行照明：

  • 全局灯光

  • 寻路/关键路径灯

  • 游戏玩法/战斗遭遇战灯

  • 细节和情绪照明

1. 全局灯

从任何全局关键光和补光灯的初始通道开始：天光、环境光和任何天空盒/大气。

如果您使用任何发射材料作为主灯，例如一条发光的熔岩河，然后也配置这个光源。

如果您想要的外观涉及烘焙间接光，请立即运行初始低质量光烘焙。一些奇特的全局照明解决方案（例如 UE5 Lumen）可能能够仅用一盏类似日光的定向光照亮整个游戏世界。

2. 寻路/关键路径灯

接下来，确保沿线的所有重要入口/出口 关键路径 从所需的进近处照明良好。您可能需要使用本地光源，例如在门口旁边放置一个灯具。

建立一个层次结构。重要的大型出口应该有更重要的照明，而次要空间应该有较暗的、不太集中的灯光。

3.游戏灯

预示遭遇战的照明（敌人逼近、战线、可能的策略和侧翼）

照明以突出谜题元素并建议探索区域

4. 细节和气氛照明

微调和调整一切，但不要花太长时间，它可能看起来足够好，如果你调整太多，你可能会破坏前一个通道的效果。

常见的照明设计问题

光烘烤后阴影太暗？

您的反照率纹理可能太暗。如果纹理太暗，那么无论您的灯光有多亮，任何产生的光反射都会很微弱。除非您有特定的风格或技术原因需要这样做，否则您的主要漫反射纹理通常应在 50-100% 的亮度范围内。看 “环境纹理价值” 作者：Rogelio Olguin，以及旧的 Epic Games UDK 文档 《Epic Games 纹理指南》.

要审查...

灯光 赋予场景视觉深度，让玩家能够测量距离并区分形状。因为它具有非常重要的游戏功能，所以您应该这样做 照明通行证 在关卡设计过程的早期，在适当的艺术通关之前。

视频游戏照明技术是一种近似现实生活中光线的粗糙幻觉。许多游戏灯光效果实际上并不使用任何照明系统。

有4种核心光源类型： 环境光、定向光、点光和聚光灯。 这些灯可以是 静态的 或 动态， 和 有动力 由可见的 灯具 或 没有动力 没有任何可见的来源。

最常见的照明理论是三点照明，它定义了 关键灯, 补光灯， 和 边缘灯 基于光线对相机和拍摄对象的方向。

要照亮一个关卡，请逐步进行多个照明通道。首先进行全局照明，然后进行寻路照明，然后进行游戏玩法照明，最后进行最后的细节或情绪照明。
