社区

如果您对 《雷神之锤》 地图和模组感兴趣， 我们强烈建议加入 《雷神之锤》 社区。 这是最好的学习方式！经验丰富的人可以为您的工作提供技术帮助、建议和反馈。有些人甚至已经修改 《雷神之锤》 20 多年了。

  • Quake Mapping Discord 是最大的公共社交中心，适合初学者，有频繁的单人游戏设计活动和社交机会

    • 对于《雷神之锤 2》，请尝试 Map-Center Discord 相反。

    • 对于多人《雷神之锤》，请尝试 QuakeWorld.nu 和 Quake.World Discord 相反。

  • Slipseer 是一个新的单人地图制作者中心，提供下载、资源和免费文件托管服务。

  • Quaddicted 是主要的单人游戏《雷神之锤》地图存档，目前正在进行大规模重建（截至 2024 年）。

  • 功能消息板 是运行时间最长的《雷神之锤》关卡设计网络论坛，但可能不像 QM Discord 或 Slipseer 那样对新手友好

编译 《雷神之锤》 地图

要玩 《雷神之锤》 地图，您必须 编译 （烘焙、打包）可编辑的 .MAP 为可播放的优化 .BSP 文件。您将需要工具和图形用户界面 (GUI)。

  • 编译工具： 埃里克工具 （已包含在 《雷神之锤》 Level Design 入门套件中）

    • qbsp.exe 构建 3D 网格 + 检测“泄漏”（视频教程：如何修复泄漏)

    • vis.exe 计算 PVS 遮挡剔除 优化地图性能

    • light.exe 烘焙光照贴图和阴影，使 灯光 发生

      • 强烈推荐观看 light.exe 示例 + 光.exe 文档

  • 编译器图形用户界面： 选择其中之一 TrenchBroom的内置编译窗口 或 Necro 的编译 GUI （两者都已包含在套件中）

资源/纹理

在《雷神之锤》模组文化中，从其他地图和模组中提取模型、声音和纹理通常被认为是正常且可以接受的，只要您 归功于原作者.

但是，如果您从 .BSP 中提取纹理和资源，则通常会得到完整套件的不完整部分。它可能会很难使用。在这些情况下（例如 Makkon 集） 您最好找到完整的官方公共纹理版本。

纹理 WAD

《雷神之锤》 纹理集合存储在 .WAD 文件中。要在关卡中使用纹理，请下载 .WAD，然后使用关卡编辑器将 WAD 路径添加到 .MAP 文件。 （注意：这与 《毁灭战士》（Steam） WAD 不同。《雷神之锤》 WAD 仅适用于地图纹理。）

将地图编译为可播放的 .BSP 文件时，所有使用的纹理都会自动直接嵌入到 .WAD 文件中。您可以使用类似的工具从编译的地图中提取纹理 BSP2WAD.

  • 原型WAD Khreathor 的作品对于草稿和原型设计非常有用。

  • 雷神之锤101 WAD 包含所有 《雷神之锤》 纹理在一个集合中，或者您也可以直接下载 每个贴图使用的纹理 如果你想严格遵守传统主题。

  • 无赖 是凯尔设置的中世纪图书馆主题纹理

  • 马克孔 是当今社区 《雷神之锤》 地图中使用的最流行的纹理集

纹理 WAD 档案

  • Quaddicted WAD 存档 包含许多 .WAD，但有些无组织。

  • 惊天动地 有一堆类似的无组织的 WAD 文件

  • Slipseer 有一个较小但不断增长的组织更好的 WAD 文件存档

WAD工具

  • 最常见的 WAD 创建/编辑工具是 德克萨斯墨西哥 和 沃利，它可以为您轻松浏览WAD并转换纹理。

  • 如果您要经常修改 WAD，您可以通过命令行界面 (CLI) 进行批处理来自动化 WAD 构建过程： 帕克曼.

  • 使用 TexMex 或 BSP2WAD 从编译的 .BSP 贴图文件中提取纹理

  • 将纹理码垛以适应 《雷神之锤》 的固定 256 调色板是很棘手的，因为某些颜色被保留为“全亮”颜色。您可能还需要 EricW 的 “去富亮” 工具来删除这些全亮像素。

天空盒

雷神之锤有两种不同的天空系统。 1996 年最初的天空系统是一个两块面板纹理，其自身存在视差。较新的 《雷神之锤》 引擎支持更标准的立方体贴图样式天空盒，该天空盒由 6 个静态 .TGA 纹理组成，每个纹理对应于天空盒的一侧。

模组/开发套件

Mod 添加了新的功能和特性，供地图绘制者在其关卡中使用。下面列出了单人游戏《雷神之锤》地图制作者使用的常见《雷神之锤》模组和工具包。

当您发布地图或模组时， 将核心模组文件与您的地图捆绑在一起 制作一个独立的 .ZIP，没有外部依赖项。例如：如果您使用 Copper 制作 mod，则应将 Copper mod 文件包含在 ZIP 中。大型模组包括精简版 “开发套件” 你可以使用，它只有核心模组文件。

碱性

碱性 是最近的“基地”主题模组/工具包，添加了科幻主题的怪物和武器。

  1. 安装 最新版本的碱性 （可用的最小开发套件）

  2. 添加 /alkaline/ mod 文件夹和 alkaline.fgd 在你的地图编辑器中

  3. 阅读包含的内容 docs.html 碱性测绘手册

铜

铜 是一个极简主义的“精致”模组，它重新平衡了原版《雷神之锤》的玩法，并为地图制作者添加了生活质量功能。它有利于“香草+”的感觉，没有香草虫。

  1. 安装 最新版本的铜

  2. 添加 /copper/ mod 文件夹和 copper.fgd 在你的地图编辑器中

  3. 阅读 官方铜 - 映射注释 获取信息和建议

程序转储

程序转储 是最近的开发套件模组，添加了新的地图功能，但没有添加任何新的怪物或武器，它仍然基本上与旧的香草雷神之锤游戏相同。

然而，由于它公开了如此多的功能，因此地图绘制者可以添加一些新的游戏功能，而无需在 《雷神之锤》C 中进行编码。

造型

有关 3D 艺术工具的列表，请参阅 工具

要制作自定义怪物、武器、物品或道具，您必须导出 3D 模型。

原始的 《雷神之锤》 模型存储在 .MDL 文件位于名为的子文件夹中 /progs/ 。 MDL 格式反映了 1996 PC 的内存限制：

  • 最大三角形：2048；最大顶点数：1024

  • 纹理：8 位调色板；最大 UV：1024

  • 最大顶点动画帧：10 FPS 时为 256

有关 MDL 文件格式和解析的更多信息，请参阅 《雷神之锤》 MDL 文件格式规范

2021 年重新发布的版本以及更新的风扇引擎添加了更多现代元素 MD5（毁灭战士 3 格式） 文件支持具有更高的内存限制和骨骼动画支持。

  • 模型数据存储在 .md5mesh 文件

  • 动画存储在 .md5anim 文件；可以为一个模型加载多个 .md5anim 文件

  • 纹理是 .tga, 结合 .mtr 材料定义文件

  • 适用于 2021 年重新发布的引擎（“Kex《雷神之锤》”）、《雷神之锤》spasm-Spiked 和 FTE
