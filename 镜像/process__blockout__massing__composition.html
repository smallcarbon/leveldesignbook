构图

关卡中形状的整体视觉排列和组织；地标、远景、途径；另外，为什么“引导线”不是真实的

什么是组合？

构图 是关卡的整体视觉组织。有两种类型的组合：

  • 空间构成： 核心整体大图3D排列 群众。我们无法保证特定的视图，而是考虑了各种视角和位置。

    • 这是建筑、室内设计和雕塑等 3D 艺术中实践的“构图”。

  • 镜头构成： 从特定位置的特定角度，关卡在玩家的 2D 相机框架/屏幕中的外观如何。

    • 这是绘画、绘画、摄影和电影等二维艺术中实践的“构图”。

许多关卡设计教程和资源都声称特定的镜头构图非常重要。由于许多原因，我们意见不一致，并且对构图有不同的定义。相反，我们认为 空间构图比镜头构图重要得多。

空间构图

空间组织包括建立一个空间 关卡制度，其中关卡的某些部分应该比其他部分更重要。层次结构的顶部是关键地标、焦点和中心区域， 对比 与周围环境有意义地结合在一起，并锚定构图的其余部分。建立空间对比的一些方法：

  • 高度。 拥挤空间中的高大塔，或开放空间中的小矮物体。

  • 密度/分布。 更宽的开放空间，周围环绕着更小、更窄的结构。

  • 方向。 脱离周围网格的有角度的物体。

  • 形状。 附近有很多长方形的东西吗？尝试在那里放一个圆形的东西。

在上图中，请注意中央立法院如何 昌迪加尔议会宫 勒·柯布西耶（Le Corbusier）（左下）在其平面图中使用了几种对比类型。它很大，是一个盒子里的圆圈，并且以与网格的其余部分分开的倾斜角度/轴旋转。所以感觉对于这座建筑的功能来说是非常重要和至关重要的！如果这是一个关卡，那么主要的场景或最终的 Boss 战斗可能会发生在这个圆形房间里。

房间偏离中心，几乎触及东墙。但它并不感觉偏离中心，不是吗？这是因为焦点在空间组织内创建了新的中心。想象一个来自重要区域的影响力区域；它有点“拉动”周围构图的其余部分。 （请参阅右下角的图表，了解焦点如何影响我们感知空间“中心”的方式。）

请记住， 层次结构取决于当地的对比和背景。 高大的物体只有被矮小的物体包围时才会显得特别。如果一件小事被大事包围，就会感觉更特别。

视线

一个 视线 是空旷空间的轨迹，提供了另一个空间的可能视图。

视线是玩家在与当前游戏状态和目标相关时使用的工具。如果玩家没有理由沿着视线看，那么他们很可能根本不会使用视线。视线是玩家可能会忽略的机会/情境工具，它不能保证玩家的行为。

即使玩家看向某个方向， 无法保证他们会记住或处理他们所看到的内容。 （想象一下阅读一本无聊的书或观看一部无聊的电影；您的眼睛可能会看到文字/图像，但这并不意味着您实际上阅读或处理了它。）

在上图中，关卡设计师 Sal Garozzo 重点介绍了 2015 版多人竞技射击游戏地图上的常见战略视线 去缓存 在《反恐精英：全球攻势》中。每个视线提供不同程度的覆盖范围和信息，用于攻击/防御炸弹点 A 区域。

然而，所有这些视线对于初次玩游戏的玩家来说，第一眼看上去会很明显吗？

不。事实上，识别和记住这些视线是学习游戏/学习地图的一部分。竞技选手必须积极研究这些视线；是的 不 一种自然的、潜意识的、明显的环顾四周的方式。 视线的存在并不意味着玩家会使用它，甚至不知道它的存在。

眺望-庇护

前景避难理论 是现实世界建筑中普遍存在的信念/设计模式，它认为人类通过寻找来导航空间 前景 （有利位置）对比 避难所 （安全区域），唤起他们与生俱来的生存本能。

我们认为前景避难是糟糕的理论（也是糟糕的科学/糟糕的艺术） 关卡设计的有限应用。相反，我们建议使用较少哲学包袱的现有概念和术语，例如视线与掩护。

有关为什么我们认为这是糟糕的理论的更多信息，请参阅 眺望-庇护.

镜头构图（谬误）

镜头构图 是关卡几何形状相对于玩家相机视角的 2D 排列。

好吧，有时其中一些很重要。视线、方法、颜色和照明、视角——有时所有这些方面都很重要。

但一些关卡设计师将镜头合成误解为关卡设计。他们认为，如果你安排关卡中的所有内容以形成精美的框架视图，那么玩家将神奇地无言地知道要做什么和去哪里。

构图不是精神控制。这是关卡设计理论中非常常见的误解，我们将用一整个章节来揭穿它。出于这个原因，我们称其为 镜头构图谬误：一种看似正确且有用的信念，但实际上是严重错误的，会导致更糟糕的理解。

截图不是关卡

首先，截图并不是一个关卡。这似乎是显而易见的，但许多关卡设计师似乎已经忘记了。

相反，屏幕截图是游戏在特定位置、角度和时间的图像。 截图是一种摄影行为.

照片是否捕捉到了某个地点或时间的客观事实？不。摄影师是选择地点和时间来创作图像的艺术家。即使是白盒的开发概述屏幕截图也会强调或淡化某些元素。照片是一种艺术诠释，捕捉 只有一方面 真实的人、地点或事物。

没有照片可以取代参观某个地方或与某人会面的体验，也没有屏幕截图可以取代试玩测试。

更重要的是，无法保证玩家会想象相同的取景或构图。 播放器永远不会播放您的屏幕截图。

有关摄影和意义长达数十年的争论的更多信息，请参阅 关于摄影 作者：Susan Sontag（关于照片和美国政治）或 露西达相机 罗兰·巴特（Roland Barthes）（关于照片和死亡）。

引导线（都是废话）

当前关卡设计的趋势是通过绘制“引导线”来分析屏幕截图，这些“引导线”似乎指向/引导玩家的眼睛看向远处的地标。

然而我们认为 引导线实际上并不有效，而且，引导线在它们自己的逻辑中甚至没有意义。

引导线误解了人类的凝视、感知和信息处理。 视网膜扫描和 视线追踪 技术展示了 2D 视觉构图的引导线和其他传统原则如何不像我们想象的那样“引导眼睛”。即使他们这样做了，也不能保证玩家确实处理了视觉信息。

几乎是你建造的每条走廊 似乎会在远处汇聚。这种现象称为 透视缩短， 在平坦的 2D 图像中创建深度错觉。使用等距或平行投影的游戏使引导线变得不可能。的 透视画的艺术史 很有趣，但这不是关卡设计。

讽刺的是，引导线会误导关卡设计师 相信镜头构图、环境美术和布景是核心关卡的设计问题，而不是 指标, 遭遇战设计, 或 试玩测试。即使引导线有效（实际上没有），您也会在关卡设计过程接近结束时进行。首先注重水平的基础基础。

如果你真的费心进行试玩测试，引导线通常会失败。 您会发现玩家有 99% 的几率会看向其他地方，有 1% 的几率会朝预期方向看 0.5 秒。 截图不是关卡，没有人玩过截图。

引导线是 自我实现的预言 当您将游戏摄像机移动到特定位置以构建特定视图时。 当然截图要遵循构图规则！那是因为你就是这样写的！然后你在上面画了线！ 这是摄影，不是关卡设计。咨询 “黄金比例”梗 关于自我实现的构图模式的荒谬。

这不是人们导航空间的方式。 人们利用模式识别、先验知识和文化联想在头脑中构建思维导图。 寻路与引导 是移动和使用环境的复杂主动过程，而不仅仅是从一个角度观看关卡中的一个点。

在空间内构建视图称为摄影，这是人类练习和发展的一项技能。这不是一个自然的潜意识过程。

镜头构图对于摄影、电影和其他静态平面 2D 媒体更有意义 作者对镜头的控制力很强，观众可以特别关注整个图像。然而，对于玩家可以在错误的时间将相机移动到错误的位置的交互式环境来说，它没有什么意义。 视频游戏关卡是一个地方，而不是绘画、照片或电影。

对于 更好的方法 影响/“指导”玩家，请参阅 寻路与引导.

视线与引导线

视线和引导线似乎是相同的。它们都与追踪玩家从摄像机角度看到的内容有关。

然而，视线和引导线强调的东西截然不同：

  • 视线： 空旷空间的轨迹，提供了另一个空间的可能视图

    • 玩家在与当前游戏状态和目标相关时使用的工具

    • 我们想再次强调：

      • 视线由空旷的空间组成

      • 视线是情境工具，不能保证行为

  • 引导线： 假想的现象，墙壁和线性布景似乎在远处汇聚，下意识地引导玩家去往哪里

    • 没有人在这样的空间中航行

    • 没有人玩这样的电子游戏

    • 不要为任何人制作关卡

要审查...

  • 空间构图 是关卡中 3D 质量的整体组织。

    • 层次结构 是指某些事物看起来比其他事物更重要，通常是因为它们在大小、形状、分布或角度方面存在对比。

      • 地标

      • 显着性

    • 视线 是玩家可以用来查看另一个区域的空白空间线。

      • 维斯塔

      • 方法

  • 镜头构图 这是一种有点扯淡的理论，认为玩家的观点决定了他们的行为，而不是玩家对空间的实际理解。

    • 屏幕截图不是关卡；而是关卡。 关卡是玩家使用的空间

    • 引导线 是脑毒，不要使用它们

    • 不要将引导线与视线混淆，它们实际上是真实的

进一步阅读

我们在关卡设计中反对引导线/镜头构图的立场目前在游戏行业中还不是常态。虽然我们认为 99% 是脑毒，但其他可信的设计师不同意：

  • “环境设计作为空间摄影：理论与实践” 作者：Miriam Bellard，Rockstar North（GDC 2019）介绍了 Rockstar 在侠盗猎车手 5 关卡中的电影环境构图方法。贝拉德提出了一种采用三分法叠加的演练方法，并强调体现空间的可供性/显着性模型。她还“打破了”主导线和小细节，同时认为它们仍然有用。总的来说，这可能是关于镜头构图最“平衡”的行业讨论，但也许只是不要喝太多 evo psych / 形状心理学酷援助.
